## 🚀 Why MicroTaskX?

- Micro‑RTOS kernel with zero dynamic memory
- Priority-based scheduler
- SmartSleep (AVR + ESP32)
- Real-time CPU profiling
- Macro timers (EVERY_MS / EVERY_HZ)
- PID, Hysteresis, Filters, CRC8
- Flicker-free LCD rendering
- Hardware utilities (fast toggle, debounced buttons)
